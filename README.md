# Portfolio

This is my personal portfolio application. Built with React.

[Live Demo](https://www.dougschallmoser.com/)


## Installation

To run this application in your local environment, please follow these steps:

```
1. Clone this repository
2. Run 'npm install'
3. Run 'npm start'
```
